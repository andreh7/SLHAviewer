slhaviewer

A Java application to view SLHA (SUSY Les Houches Accord) files

View project on GitHub

What is it ?

A java application to view SUSY Les Houches Accord (SLHA) files, released under the Apache License:

How to build it ?

The easiest way is building it with Apache maven: 
$ git clone https://github.com/andreh7/SLHAviewer.git
$ cd SLHAviewer
$ mvn package
you should then get a file slhaviewer.jar in the target/ subdirectory.

How to run it ?

If you have an SLHA file example.slha run slhaviewer as follows: 
$ java -jar slhaviewer.jar example.slha
